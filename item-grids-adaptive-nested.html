Nested

As with the regular grid the adaptive grid can be nested. This example shows two units pam-Unit="1-1 medium:1-2" that each contain a adaptive grid.

1 ➝ 2
1 ➝ 2
3 ➝ 2
3 ➝ 4
3 ➝ 4
